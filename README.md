# Brightcove Video Connect for SharePoint

Brightcove Video Connect for SharePoint enables management of Brightcove videos and playlists from within SharePoint, including content upload, metadata editing, and embedding videos in SharePoint pages via App Parts and Web Parts.

Supported on SharePoint 2013/2016/2019 On-premise and (historically) SharePoint Online via the SharePoint Add-In framework.

> [!WARNING]
> **SharePoint Online (M365) customers: this connector is being retired.**
>
> Microsoft is retiring the SharePoint Add-In framework that this connector is built on. The legacy connector will stop working in SharePoint Online (M365) on **2 April 2026**.
>
> SharePoint Online customers must migrate to the **SPFx-based Brightcove Connector** — see [Brightcove Video Connect for SharePoint (SPFx)](https://integrations.support.brightcove.com/sharepoint/index.html) on the Brightcove support site for documentation and downloads.
>
> SharePoint **On-premise** customers may continue to use this connector.
